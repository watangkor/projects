SVG 指可伸缩矢量图形 (Scalable Vector Graphics); SVG 用来定义用于网络的基于矢量的图形; SVG 使用 XML 格式定义图形; SVG 图像在放大或改变尺寸的情况下其图形质量不会有所损失; SVG 是万维网联盟的标准; SVG 与诸如 DOM 和 XSL 之类的 W3C 标准是一个整体;

矩形 rect; 圆形 circle; 椭圆 ellipse; 线 line; 折线 polyline; 多边形 polygon; 路径 path

1. 矩形：

  • rect 元素的 width 和 height 属性可定义矩形的高度和宽度
  • style 属性用来定义 CSS 属性
  • CSS 的 fill定义矩形填充颜色(常规); CSS 的 stroke-width定义边框宽度; CSS 的 stroke定义边框颜色
  • x,y定义矩形到父元素左侧和顶端的距离
  • opacity:定义整个元素的透明值;fill-opacity定义填充颜色透明度;stroke-opacity定义边框颜色的透明度
  • rx,ry 产生圆角

圆

  • cx 和 cy 属性定义圆点坐标。默认：(0, 0)r 属性定义圆的半径
  • stroke="black":边框颜色,stroke-width="2":边框宽度;fill="red":背景色.注：即可以用style,也可单独作属性
  • ellipse椭圆，多：rx,ry用来定义水平和垂直半径

线条

  • x1,y1对应线条的起始点;x2,y2对应线条的结束

polygon 多边形：用来创建含有不少于三个边的图形

  • points对应角的坐标

polyline 折线：创建仅包含直线的形状
